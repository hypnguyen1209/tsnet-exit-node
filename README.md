# tsnet-exit-node

```
flag needs an argument: -h
Usage of tsnet-exit-node.exe:
  -dir string
        State Dir (default "tsn-state")
  -h string
        Host Name
  -k string
        Auth Key
  -t string
        tag:warp-vpn (default "tag:warp-vpn")
  -u string
        Control URL https://
  -x string
        Socks5 Proxy Address (example: 127.0.0.1:1080)
```
