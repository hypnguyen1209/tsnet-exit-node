# tsnet-exit-node

```
flag needs an argument: -h
Usage of tsnet-exit-node.exe:
  -dir string
        State dir (default "tsn-state")
  -h string
        Host Name
  -k string
        AuthKey
  -t string
        tag:warp-vpn (default "tag:warp-vpn")
  -u string
        Control URL https://
  -x string
        Socks5 Proxy Address (default "127.0.0.1:1080")
```
